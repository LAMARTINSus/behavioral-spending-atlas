# Behavioral Spending Atlas — Rebuild v3

This rebuild includes:
- 40 refined questions
- improved scoring matrix
- social influence and FOMO layer
- premium-style report screens
- pilot data capture (name, most-used social platform, optional username, gender, age range, email)
- creator attribution via `?creator=name`

## Run locally
```bash
pip install -r requirements.txt
python app.py
```

## Render
Use:
- Build Command: `pip install -r requirements.txt`
- Start Command: `gunicorn app:app`
